## Что же делает наш "Гроссмейстер" Gulp?

Этот сборщик работает с точностью швейцарских часов и хитростью самого Бендера:

- **Сжимает HTML** в режиме `production` — как Бендер сжимал в объятиях Паниковского!
- **Удаляет комментарии** из HTML — потому что в Нью-Васюках лишние слова не нужны!
- **Собирает SCSS файлы**, добавляет вендорные префиксы — как гроссмейстер готовит дебютную репертуарную книгу!
- **Сортирует медиа-запросы** — расставляет всё по своим местам, как шахматные фигуры на доске!
- В режиме `production` **сжимает CSS** и делает копию без сжатия — на всякий случай, если комиссия попросит оригиналы!
- **Конвертирует шрифты** в `.ttf`, и из `.ttf` в `woff/woff2` — потому что даже в шахматах важна красота букв!
- **Создаёт файл** для подключения шрифтов. Данный файл создаётся по такому пути: `src/scss/theme/_fonts.scss`, выглядит это так:

```scss
@font-face {
  font-family: Inter;
  font-display: swap;
  src: url('../fonts/Inter-Bold.woff2') format('woff2');
  font-weight: 700;
  font-style: normal;
}
```

### Тайные псевдонимы (Alias)
`@img` — секретная папка с изображениями для наших шахматных иллюстраций!

### ⚠️ ВНИМАНИЕ, КОМБИНАТОРЫ!

> Если в папке `src/scss/theme` уже есть файл `_fonts.scss` — при добавлении новых шрифтов **НЕОБХОДИМО УДАЛИТЬ** старый! Не волнуйтесь, уважаемые, при повторном запуске Gulp всё сделает как по нотам — новые шрифты сконвертирует и создаст новый файл `_fonts.scss`. Это не обман, а оптимизация!

## Что ещё умеет наша шахматная сборка?

- **Сжимает изображения** и конвертирует в `.webp` — экономит трафик, как Бендер экономил слова!
- **Копирует папку `/static`** — всё самое ценное переносит в финальную сборку без лишних вопросов!
- **Создаёт "svg спрайты"** командой `npm run svgSprite` — как фокусник достаёт кроликов из шляпы!
- **Чистит папку с проектом** перед каждой сборкой — порядок превыше всего, даже в Васюках!
- **Архивирует проект** командой `npm run zip` — готовит доставку для заказчика с именем проекта!
- **Запускает сервер** с автоматической перезагрузкой — чтобы вы не теряли ни секунды драгоценного времени!
- **Выгружает на хостинг** командой `npm run deployFTP` — доставляет проект в мир, как почтовый голубь!
- **Поддержка шрифтов с хитрыми названиями** с 18.08.2023 — даже "`Inter-Regular[ |-|_|__][I|i]talic`" будет обработан правильно, как Бендер обрабатывал сложные ситуации!
- **Умная конвертация шрифтов** — из `src/fonts` в `dist/fonts`, без лишней суеты и нервотрёпки!

## А что там у нашего "Ферзя" WebPack?

Этот гений JavaScript работает с изяществом шахматного мастера:

- **Обрабатывает JavaScript файлы** — как гроссмейстер просчитывает комбинации на много ходов вперёд!
- **Поддерживает модульное подключение** `import/export` — потому что даже в Васюках важна организация!
- **Создаёт множество JS файлов** — для разных страниц, как разные партии в турнире!
- **Умный импорт** — не нужно писать расширения, WebPack всё поймёт, как Бендер понимал людей!

```javascript
import * as helpers from './helpers' // './helpers/index.js' — Бендер всегда знал короткие пути!
```

- **Позволяет писать на ES6+** — современный синтаксис для современного Нью-Васюков!
- **Сжимает код в production** — убирает всё лишнее, как фальшивомонетчик убирает следы!
- **Мультифайловая сборка** с 11.2023 — сколько файлов создадите, столько и получите!
